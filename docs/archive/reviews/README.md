# Archived Review Artifacts

This directory preserves historical review artifacts that were previously kept
at the repository root. Older session logs may still reference the original root
paths; the files now live here to keep the root focused on project entrypoints,
configuration, and source directories.

## Index

- `tier4-proposal-r1/` - Tier 4 proposal review round 1 artifacts, including
  product, technical alignment, and consolidated reviews.
- `pr-d-phase-b/` - PR D Phase B peer, alignment, adversarial, and
  consolidation review rounds.
- `pr-e/` - PR E peer, alignment, adversarial, and consolidation review rounds.
