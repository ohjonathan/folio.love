Gemini's concern is real and preserved as a should-fix (see Should-Fix Findings §A-1/A-2/A-3/A-4 below), but it doesn't justify re-scoping the deliverable. The operator-level resolution path is to amend the proposal doc with a "Shipping Plan" section that formalizes slice-ability — a parallel edit, not a spec-under-review change.
